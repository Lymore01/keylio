# Contributing to Keylio Auth

Thank you for your interest in contributing to Keylio Auth. This guide will help you get started with our monorepo project.

## Getting Started

1.  **Clone the repository**
2.  **Install dependencies**: Run `pnpm install` (or your preferred package manager) in the root directory to install dependencies for all workspaces.
3.  **Development**: Check the root `package.json` for available scripts (e.g., `pnpm run dev`, `pnpm run build`).

## Project Structure

As a monorepo, this project contains multiple packages:

- `apps/`: Contains the main applications.
- `packages/`: Contains shared libraries and utilities.

## Submitting Changes

1.  Create a new branch for your feature or fix.
2.  Ensure all tests and linting checks pass.
3.  Submit a Pull Request describing your changes.
