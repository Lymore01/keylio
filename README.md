## About the Project

Keylio Auth is a framework-agnostic authentication library for TypeScript. It offers a simple yet extensible set of features, with a plugin ecosystem for adding advanced functionalities quickly. Focus on your app while Keylio handles auth.

### Why Keylio

Authentication in TypeScript can be complex. Keylio simplifies it with a clean, modular design inspired by popular libraries, but built to be easier to extend and integrate.
